Demo Overview

This demo website presents the main idea and experimental results of our CVPR 2026 paper, “Breaking Sensor-Text Alignment: Cross-Modal Attack on Contrastive Multimodal Human Activity Recognition.” The paper studies a security weakness in contrastive multimodal HAR systems, where wearable sensor signals and natural-language activity descriptions are aligned in a shared embedding space.

The demo shows how a small text-side embedding perturbation can reduce the correct sensor-text cosine similarity, increase contrastive loss, and cause semantic drift. These effects can make the model match a sensor pattern with an incorrect activity description. The website also includes an interactive perturbation visualization and result graphs for WISDM, MotionSense, HHAR, and WESAD.

How the attack works

The target model aligns sensor windows and natural-language activity descriptions in a shared embedding space. The attack perturbs the text embedding during inference, making the correct sensor-text pair less similar and pushing the system toward incorrect activity interpretation.

📡

Sensor Window

Wearable signals from phones, smartwatches, or physiological sensors.

🧠

Sensor Encoder

Maps time-series features into a 64-dimensional embedding.

💬

Text Encoder

Encodes semantic activity descriptions using a language model.

🎯

Attack δ

Adds a bounded perturbation to the text embedding at inference time.

⚠️

Misalignment

Similarity drops, loss rises, and the predicted description changes.

Experimental Results

The graphs summarize the effect of the cross-modal embedding poisoning attack. The PGD-step plots show how many optimization steps are needed for successful and failed attacks across perturbation values. The norm plots show the relationship between L∞ and L2 perturbation magnitudes, where stronger perturbations generally lead to more reliable attack success.

Traditional HAR vs. Contrastive HAR

Traditional HAR directly classifies activities, while contrastive HAR aligns sensor embeddings and text embeddings in a shared space.

Cross-Modal Attack Overview

The attacker injects a bounded perturbation into the text embedding during inference, causing target sensor-text misalignment.

Similarity and Loss Changes

The attack reduces adversarial similarity and increases contrastive loss across datasets and activities.

Key takeaway

Contrastive sensor-text systems are powerful because they connect wearable signals with human-readable descriptions. However, this same alignment mechanism creates a new attack surface. A small embedding-level change can weaken the correct match, increase loss, and cause semantic misinterpretation.
